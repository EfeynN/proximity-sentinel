# Changelog

All notable changes to **Proximity Sentinel** will be documented in this file.

This project follows a structured, milestone-based development model.
Versions below 1.0.0 may introduce breaking changes.

## [v1.0.0] – Production Release (Planned)
### Added
- Production-ready detection and scoring pipeline
- Stable enforcement thresholds
- Hardened false-positive protection
- Finalized admin review workflows
- Audit-grade logging and traceability

### Changed
- Optimized risk score decay model
- Refined behavior signal weighting
- Hardened client-server trust boundaries

### Security
- Full security review completed
- Threat model validated against real-world scenarios
